### FOR TERMUX/SSH/UBUNTU

```bash
> apt update
> apt upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> git clone https://github.com/ZeroChanBot/Zeroo
> cd Zeroo
> npm install
```

### FOR WINDOWS/RDP/VPS

* Click & Install [`GIT`](https://git-scm.com/downloads)
